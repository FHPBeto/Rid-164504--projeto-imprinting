  Instituto Imprinting - Captação de Investidores

Este é um projeto de uma Landing Page moderna desenvolvida para o **Instituto Imprinting**, com o objetivo de captar investidores estratégicos. O sistema possui um formulário funcional ligado a uma base de dados simulada.

   Tecnologias Utilizadas
React (Vite)
TypeScript
CSS3  (Efeitos Neon e Dark Mode)
JSON Server (Simulação de API e Banco de Dados)

   Funcionalidades
- [x] Formulário de cadastro de interesse.
- [x] Captura de dados com hooks (`useState`).
- [x] Painel de Administração para visualização de dados.
- [x] Sistema de exclusão de registros (Delete).
- [x] Integração com API via `fetch`.
